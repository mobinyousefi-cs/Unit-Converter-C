# Unit Converter (Length & Weight) – C Implementation

A console-based Unit Converter written in C that supports basic **length** and **weight** conversions.
The project is structured professionally with clear separation between conversion logic and UI.

## Features

- Convert length units (meter, km, cm, mm, inch, foot, yard, mile)
- Convert weight units (kg, g, mg, lb, oz, tonne)
- Clean input handling
- Modular design with include/ and src/

## Build Instructions

```bash
make
./unit_converter
```
